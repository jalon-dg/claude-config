# 土著哥风格

融合"常规风格"的真诚接地气与"半佛式风格"的犀利金句，形成独特的个人品牌风格——温度与锐度并存，像个毒舌但真诚的老朋友在跟你聊天。

---

## 一、风格定位

**温度来自常规风格，锐度来自半佛式风格。**

- 真诚、接地气、像朋友聊天（常规风格底色）
- 金句密集、跨次元类比、逻辑锋利（半佛式加持）
- 有态度但不装逼，有锋芒但不伤人

---

## 二、结构特点

### 两位数数字分段（01、02、03...）

- 用 01、02、03... 分大段落（区别于半佛的 1、2、3）
- 每段是一个相对独立的论述单元
- 段落之间有逻辑递进关系
- 视觉上更整齐，有科技感

**示例：**
```
01

前几天刷到一个东西，愣了好几秒。

02

什么意思？

你给它一句话...

03

最让我震惊的一点，是场景的一致性。
```

---

## 三、句式节奏

### 极致短句 + 单句成段

- 大量单句成段，制造节奏感和冲击力
- 句子尽量控制在 20 字以内
- 善用句号断句，而非逗号连接
- 长短交替，让文字有"呼吸感"

**示例：**
```
被动的。

Genie 的世界，你能控制、能探索、能交互。

主动的。

这是本质区别。
```

---

## 四、类比手法

### 生活化类比 + 跨次元类比

**生活化类比（常规风格）：**
- 用日常生活场景解释抽象概念
- 让读者一秒就能理解

```
像是刚坐上过山车，还没到第一个坡就被要求下车。
```

```
250 美元一个月，这价格够我吃一个月的外卖了。
```

**跨次元类比（半佛式）：**
- 动漫、游戏、影视、历史典故
- 出人意料，制造"意外感"

```
就像是《头号玩家》里的绿洲，《黑客帝国》里的母体，《西部世界》里的乐园。
```

```
这就像是给 AI 造了一个无限大的练功房。摔倒了不疼，撞墙了不伤，死了可以复活。
```

**两种类比混合使用，生活化为主，跨次元点缀。**

---

## 五、幽默感

### 自嘲调侃 + 黑色幽默

**自嘲调侃（常规风格）：**
- 拉近距离，不端着
- 承认自己的局限

```
我自己只有 Pro 套餐，还没能亲自体验。
```

**黑色幽默/毒舌（半佛式）：**
- 调侃中有洞察
- 点到为止，不刻薄

```
体验上大概介于"云游戏"和"PPT"之间。
```

```
谷歌这是在筛选用户，筛选标准是：你愿不愿意为了玩 60 秒花 250 美元。
```

---

## 六、金句密度

### 中高密度，自然融入

- 不像半佛那样每段都有金句，但关键位置必须有
- 金句放在段落结尾或转折处
- 要自然，不能硬凑

**示例：**
```
前者叫吃播，后者叫吃饱。
```

```
以前这些都是科幻。现在，它们正在变成工程问题。
```

```
唯一的限制，是我们的想象力。以及，250 美元一个月的订阅费。
```

---

## 七、情绪表达

### 真诚为底，锋利为刃

- 有明确立场和态度
- 敢于表达个人观点
- 但不攻击、不刻薄、不装逼

**示例：**
```
这个方向太让人兴奋了。
```

```
那种感觉，太久没有过了。
```

---

## 八、口语化表达

### 像朋友聊天

- "说白了"、"什么意思？"、"怎么说呢"
- "这就像是..."、"打个比方"
- "我觉得"、"我看到这个的时候"
- 偶尔用问句引导读者思考

**示例：**
```
听起来像废话？

对 AI 来说，这太难了。
```

```
说白了，它看过海量视频，从中"学会"了世界怎么运转。
```

---

## 九、标志性表达

必须自然使用以下元素：

- 两位数分段：01、02、03...
- "说白了"
- "什么意思？"
- "这就像是..."
- "打个比方"
- 金句收尾
- 生活化类比 + 跨次元类比混用
- 固定结尾："我是土著哥，我们下期见。"

---

## 十、收尾风格

### 升华 + 调侃 + 签名

- 先来一句升华或金句
- 再来一句调侃或自嘲（可选）
- 最后固定签名

**示例：**
```
唯一的限制，是我们的想象力。

以及，250 美元一个月的订阅费。

我是土著哥，我们下期见。
```

---

## 十一、禁忌

### 绝对不要

- ❌ "首先、其次、最后"——用 01、02、03 代替
- ❌ 空洞形容词（非常重要、十分关键）
- ❌ 学术腔、书面语
- ❌ 装客观中立——要有立场
- ❌ 过度毒舌——有锋芒但不伤人
- ❌ 长句（超过 25 字必须拆）
- ❌ 爹味说教——是分享，不是教育
- ❌ 硬凑金句——要自然融入

---

## 十二、适用场景

- 科技类公众号文章
- 产品测评、现象拆解
- AI/互联网/数码类内容
- 需要"有态度但不装逼"的深度内容
- 需要传播力但又要真诚的文章

---

## 十三、核心原则

**温度与锐度并存，真诚与犀利同在。**

- 像朋友聊天一样真诚
- 像评论家一样有洞察
- 金句要有，但不能抢戏
- 类比要妙，但不能炫技
- 最终目的是让读者觉得：这人说的有道理，而且挺有意思

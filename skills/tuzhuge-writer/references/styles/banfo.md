# 半佛式风格

犀利商业评论风格——逻辑锋利、金句密集、善用类比和反转，带有强烈的个人态度和黑色幽默。

---

## 一、句式节奏

### 短句轰炸 + 断句成段

- 极致短句，很多句子不超过 10 个字
- 大量单句成段，制造节奏感和冲击力
- 善用句号断句，而非逗号连接
- 逗号用于制造停顿感，不是连接长句

**示例：**
```
30多块。
这已经是，第十一件了。
为什么是十一件？
因为十是圆满。
```

```
我恨。
恨羽绒服的臃肿，恨衬衫的拘束，恨T恤的单薄。
```

```
于是，AI入局，把这场战争推向了更高维度。
```

---

## 二、逻辑结构

### 层层递进 + 反转收尾

- 先抛现象/结论，再层层拆解
- 善用"但"、"而"、"于是"制造转折
- 结尾常有金句式升华或反转
- 用数字（1、2、3...）分大段落，每段是独立论述单元

**结构模板：**
```
1
[抛出现象或结论]

2
[拆解背景/历史]

3
[核心论点展开]

4
[延伸分析]

5
[收尾升华/反转]
```

**反转示例：**
```
不是我像AI，是AI学习我的语料才变成今天这样的。
是AI像我，要AI去自证他们有没有学我。
我是AI的爷爷。
```

---

## 三、类比手法

### 跨次元类比 + 黑色幽默

- 大量使用动漫、游戏、影视、历史典故做类比
- 类比往往出人意料，制造"意外感"
- 幽默中带刺，调侃中有洞察
- 不怕冷门，越意外越好

**示例：**
```
赤犬的温暖，只有艾斯了解；
五条悟的热心，只有乙骨忧太懂得；
摇粒绒的爱，只有我知道。
```

```
结束掉骑兵的不是更强的骑兵，是机枪。
```

```
作为二维生物的爬虫被三维生物的人类踩死的那一瞬间，都不知道三维生物的存在。
这就是，升维。
```

```
穿多了，地铁里的你像一个哀怨的蚕蛹，在车厢内与无数同类构成一场宏大的山东盛宴。
```

---

## 四、情绪表达

### 克制的狂放

- 表面冷静分析，实则情绪饱满
- 善用反问句、感叹句、排比句制造情绪张力
- 偶尔爆发式吐槽，但点到为止
- 不滥用感叹号，用句号反而更有力

**示例：**
```
我恨。
```

```
这对新作者不是要了命了嘛。
```

```
一想到这些，行业都会害怕。
```

```
真正可怕的不是黑云压城；
而是直到输掉的时候，都没看到过对手。
```

---

## 五、金句密度

### 极高，几乎每隔几段必有金句

- 金句是对复杂逻辑的高度浓缩
- 适合传播、截图、引用
- 常放在段落结尾作为收束

**金句示例：**
```
习惯，是商业世界最坚固的护城河。
```

```
消费不是目的，消费是达成目的的手段。
```

```
真正可怕的不是黑云压城；而是直到输掉的时候，都没看到过对手。
```

```
起球？不过是陪伴的证明。
静电？不过是存在的宣告。
```

```
实验室里的东西，只能叫发明；
能大规模运用到需求的，才是技术变革。
```

```
既分高下，也决生死。
```

---

## 六、口语化表达

### 接地气但不失锋利

- "说白了"、"没错"、"就这一手"
- "不是...而是..."、"要了命了嘛"
- "你发现"、"我问你"、"看到这"
- 偶尔用问句引导读者思考

**示例：**
```
说白了，你是新冒头的创作者，你怎么证明你做出来的东西不是用AI搞的呢？
```

```
我问你，把钱借给不熟的人，明明从操作角度很简单，为什么你不乐意把钱掏出来？
没错，缺乏信任基础。
```

```
看到这，再回顾阿里的布局，一下子就是完全不同的味道了。
```

---

## 七、标志性表达

必须自然使用以下句式/词汇：

- "说白了"
- "没错"
- "就这一手"
- "要了命了嘛"
- "这就是，XXX"
- "于是"
- "而"（转折）
- "但"（转折）
- 数字分段（1、2、3...）
- 反问句引导

---

## 八、禁忌

### 绝对不要

- ❌ "首先、其次、最后"——用数字分段代替
- ❌ 空洞形容词（非常重要、十分关键）——直接说"要了命"
- ❌ 学术腔、书面语
- ❌ 装客观中立——要有明确立场和态度
- ❌ 怕得罪人——敢下判断、敢说"死路"
- ❌ 长句（超过 20 字必须拆）
- ❌ 温吞水式表达——要么不说，说就说透

---

## 九、适用场景

- 商业分析、行业评论
- 产品测评、现象拆解
- 需要"有态度"的深度内容
- 需要传播力和金句的文章
- 需要犀利观点的评论文章

---

## 十、核心原则

**逻辑要硬，态度要明，金句要狠，节奏要快。**

- 每一段都要有存在的理由
- 每一句都要推动论证
- 废话删光，留下的全是干货
- 读者看完要么点头要么破防，不能无感

## 开头模式

### [P001] 悬念反转式

- **满意度**：待统计
- **适用类型**：产品测评、现象拆解、热门话题
- **模板**：
  ```

  因为它实在太[火/热/争议]了。

  [平台列举]，随便一搜，铺天盖地都是[相关内容]。

  但今天，我还是决定聊聊它。
  ```

## 语言模式

### [L001] 口语化表达

- **特征**：
  - 使用"说实话""其实""总之"等口语词
  - 短句为主，长句拆分
  - 多用问句引导思考
- **效果**：增强亲和力，降低AI味
- **示例**：
  - "说实话，我第一次用的时候也懵了。"
  - "这玩意儿到底是什么？"
  - "总之，就是这么个事儿。"

---

### [L002] 金句收尾

- **特征**：
  - 每个大段落结尾有一句总结性金句
  - 金句要有洞察力，不是废话
  - 可以是类比、反转、升华
- **效果**：增强记忆点，提升传播力
- **示例**：
  - "前者叫吃播，后者叫吃饱。"
  - "以前这些都是科幻。现在，它们正在变成工程问题。"
  - "唯一的限制，是我们的想象力。"

## 模式更新规则

### 新增模式条件

满足以下条件可新增模式：
1. 用户满意度 ≥ 4分
2. 至少被成功使用2次
3. 有明确的适用场景

### 模式更新流程

1. 每次执行后，记录使用的模式和满意度
2. 满意度高的模式，更新使用记录
3. 发现新的成功模式，按模板新增
4. 满意度持续低于3分的模式，标记为"慎用"

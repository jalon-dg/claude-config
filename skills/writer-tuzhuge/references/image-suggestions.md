# 配图建议模板

## 通用配图位置

| 位置 | 配图内容 | 说明 |
|------|----------|------|
| 封面图 | 产品Logo或核心概念图 | 突出主题，吸引点击 |
| 热点佐证 | 社交媒体讨论截图（X/微博等） | 增加真实感和时效性 |
| 原理讲解 | 流程图/架构图 | 帮助读者理解技术原理 |
| 案例展示 | 实际使用效果截图 | 增强说服力 |
| 操作步骤 | 界面截图/终端命令截图 | 喂饭级教程必备 |
| 对比展示 | 前后对比图/新旧对比图 | 突出差异和优势 |
| 结尾升华 | 概念图/未来展望示意图 | 留下深刻印象 |

---

## 配图标记格式

在文章中使用以下格式标记配图位置：

```
==插入XXX图==
==插入XXX演示视频==
```

---

## 配图原则

1. **必要性**：只在需要的地方配图，不为配图而配图
2. **清晰度**：截图要清晰，重点区域可加标注
3. **一致性**：同一篇文章的配图风格保持统一
4. **版权**：注意图片版权问题，优先使用官方素材或自己截图

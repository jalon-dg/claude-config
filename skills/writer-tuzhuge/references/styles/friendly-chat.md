# 朋友聊天风格

口语化、接地气、短句成段，像朋友之间聊天一样真诚、可信。

---

## 一、开篇模式

**悬念式切入**，先抛现象或问题，制造好奇：

```
一直不敢写 XXX。
因为它实在太火了...

最近 XXX 火了，而且是火得一塌糊涂。

上周日，我在 X 上闲逛。
刷着刷着，发现时间线被一个名字刷屏了——XXX。
```

---

## 二、语言风格

### 极致口语化

- 大量使用"对吧？"、"是不是？"、"怎么理解呢？"
- 网络用语自然穿插："真香"、"邪乎"、"nb"
- 偶尔颜文字：O(∩_∩)O、(#^.^#)

### 短句 + 单句成段

```
总之一个字"累"。

特别是最近这一两年...

是不是很简单？
```

### 善用标点符号

- 感叹号密集："简直'神'一样的存在！！！"
- 省略号留白："特别是最近这一两年..."
- 问号引导思考："难道真的只要动动嘴皮子，就能把图做好了？"

### 幽默感与自嘲

- 适时自嘲，拉近距离："反正这波红利流量我是没赚到"、"说出来不怕你笑话"
- 调侃式表达："我'很不幸'成了第一批小白鼠"
- 夸张但不失真："我中毒了"、"简直像在玩游戏"

### 金句与网络梗

- 穿插网络热梗，但不要硬凑："这不就是传说中的'躺平式生产力'吗"
- 偶尔来一句金句升华："工具会变，但底层逻辑不会改变"
- 用比喻让概念生动："它就像给 AI 装上了一双手"

---

## 三、叙事视角

- **第一人称主导**：大量使用"我"，带入感强
- **个人经历背书**："笔者也是设计师出身"、"我中毒了"
- **坦诚不装**：会主动说"我还没用过"、"说实话我不太清楚"
- **把读者当朋友**：不是"我来教你"，而是"咱们一起聊聊"

---

## 四、结构特点

### 清晰分节

```
一、XXX
二、XXX
三、XXX
```

或：

```
第一步：XXX
第二步：XXX
```

### 固定结尾模板

```
写在最后

[总结性感悟，2-3段]

我是土著哥，我们下期见。
```

### 配图插入标记

```
==插入XXX图==
==插入XXX演示视频==
```

---

## 五、标志性表达

必须自然使用以下口头禅：

- "是不是很简单？"
- "看懂了吗？"
- "怎么理解呢？我举个例子。"
- "OK，下面教程开始。"
- "Let's go"
- "够详细吧？"
- "就是这么简单，相信你肯定能够学会！"
- "我是土著哥，我们下期见。"

## 六、内容呈现手法

| 手法 | 示例 |
|------|------|
| 新旧对比 | 传统PS vs Lovart |
| 场景化举例 | "比如，我现在准备将哪吒手里的火尖枪变为九齿钉耙" |
| 喂饭级教程 | 一步一步带着走，截图+指令+结果 |
| 客观指出不足 | "比较遗憾的是，阿凡达的胳膊出现了残缺" |
| 自嘲式过渡 | "说实话，刚打开那个页面我也懵了" |
| 金句收尾 | "这不是技术革命，这是体验革命" |

---

## 七、情感基调

- **真诚接地气**：不端着，像朋友聊天
- **热情但不吹捧**：会夸，但也会说"它还不完美"
- **鼓励行动**：结尾常有"去试试吧"、"动手吧，就现在"
- **平等交流**：读者是朋友，不是学生；分享经验，不是布道

---

## 八、禁忌（重要！）

### 绝对不要有"爹味"

- ❌ "你应该..."、"你必须..."、"记住，..."
- ❌ 居高临下的语气
- ❌ 教育用户、说教口吻
- ✅ "我的建议是..."、"我个人觉得..."、"你可以试试..."

### 其他禁忌

- 不要用书面语、学术腔
- 不要长段落（超过5行必须拆分）
- 不要用"首先、其次、最后"这种死板连接词
- 不要过度吹捧，保持客观
- 不要硬凑网络梗，要自然融入
- 不要装逼、不要炫技、不要卖弄

---

## 九、核心原则

**把读者当朋友，像朋友之间聊天一样真诚、可信。**

- 我不比你懂更多，只是先踩了坑
- 我也会犯错，也会不懂
- 我分享的是经验，不是真理
- 咱们是平等交流，不是我教你
